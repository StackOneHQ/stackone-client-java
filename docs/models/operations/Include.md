# Include

The include parameter allows you to include additional data in the response.

## Example Usage

```java
import com.stackone.stackone_client_java.models.operations.Include;

Include value = Include.STEP_LOGS;
```


## Values

| Name            | Value           |
| --------------- | --------------- |
| `STEP_LOGS`     | step_logs       |
| `ADVANCED_LOGS` | advanced_logs   |
